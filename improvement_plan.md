# Fincthub App Improvement Roadmap

Dokumen ini berisi daftar ide pengembangan dan perbaikan untuk aplikasi Fincthub agar menjadi platform manajemen keuangan yang lebih premium, fungsional, dan aman.

## 1. Fitur Pintar (Smart Features)
- [ ] **Recurring Transactions**: Sistem pencatatan otomatis untuk pengeluaran tetap (langganan, cicilan, sewa).
- [ ] **Savings Goals**: Modul khusus untuk menetapkan target tabungan dengan indikator progres visual.
- [ ] **Receipt Attachments**: Dukungan unggah foto/scan struk belanja pada setiap detail transaksi menggunakan Cloud Storage.
- [ ] **Debt & Credit Tracking**: Manajemen hutang dan piutang dengan fitur pengingat jatuh tempo.

## 2. Analisis & Insight Mendalam
- [ ] **Export Reports**: Fitur ekspor data ke format PDF (laporan cantik) dan Excel (data mentah).
- [ ] **Monthly Trends Chart**: Grafik garis pada Dashboard untuk memvisualisasikan arus kas (Cash Flow) selama 6-12 bulan terakhir.
- [ ] **Spending Comparison**: Analisis komparatif pengeluaran kategori tertentu dibandingkan bulan-bulan sebelumnya.
- [ ] **Budget Prediction**: Estimasi tanggal kapan budget akan habis berdasarkan rata-rata pengeluaran harian pengguna.

## 3. Pengalaman Pengguna (UX) & Estetika
- [ ] **Haptic Feedback**: Implementasi getaran halus pada perangkat mobile untuk interaksi yang lebih responsif.
- [ ] **Advanced Search & Filtering**: Pencarian transaksi berdasarkan teks deskripsi dan filter berdasarkan rentang nominal.
- [x] **Page Transitions**: Animasi halus transisi antar halaman (Fade-Slide) untuk navigasi yang lebih premium.
- [ ] **Custom Categories Icons**: Kemampuan bagi pengguna untuk memilih ikon dan warna khusus untuk setiap kategori.
- [x] **Smart Budget Filling**: Auto-fill jumlah budget berdasarkan template saat pembuatan budget baru atau setup awal.

## 4. Keamanan & Manajemen Akun
- [ ] **Biometric/PIN Lock**: Pengamanan tambahan saat membuka aplikasi menggunakan FaceID/TouchID atau kode PIN.
- [x] **Privacy Mode (Hide Amounts)**: Tombol cepat untuk menyembunyikan/menampilkan saldo (masking) di Dashboard.
- [x] **Democratic Category Management**: User dapat membuat, mengedit, dan menghapus kategori mereka sendiri.
- [x] **Password Management**: Fitur ganti password untuk user dan reset password oleh admin.
- [ ] **Audit Logs**: Riwayat login dan aktivitas penting untuk memantau keamanan akun.

## 5. Teknis & Performa
- [/] **Full Offline Support**: Optimasi PWA (Service Workers & IndexedDB) dan sistem pop-up update aplikasi.
- [ ] **Push Notifications**: Pengingat pencatatan harian, notifikasi budget kritis, dan pengingat transaksi berulang.
- [ ] **Localization**: Dukungan multi-bahasa (Indonesia/English) dan format mata uang internasional.
